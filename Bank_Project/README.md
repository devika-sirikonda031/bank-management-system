# Bank Management System (Python + MySQL)

A simple console-based Bank Management System developed using **Python** and **MySQL**.

## Features

- Create Account
- Deposit Money
- Withdraw Money
- Check Account Balance
- View All Accounts

## Technologies Used

- Python
- MySQL
- mysql.connector library

## Project Structure

Bank_Project
│
├ create_db.py        # Create database and table
├ db_connect.py       # MySQL database connection
├ bank.py             # Bank operations
├ main.py             # Menu driven program
└ README.md           # Project documentation
