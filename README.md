# Bank Management System (Python + MySQL)

A simple console-based Bank Management System developed using **Python** and **MySQL**. This project allows users to perform basic banking operations through a menu-driven interface.

---

## Features

* Create Account
* Deposit Money
* Withdraw Money
* Check Account Balance
* View All Accounts

---

## 🛠 Technologies Used

* Python
* MySQL
* mysql-connector-python

---

## Project Structure

Bank_Project/
│── create_db.py        # Database and table creation
│── db_connect.py       # Database connection
│── bank.py             # Banking operations
│── main.py             # Main menu program
│── README.md           # Documentation

---

##  How to Run

1. Install Python
2. Install MySQL
3. Install MySQL Connector:

pip install mysql-connector-python

4. Run database setup:

python create_db.py

5. Run the main program:

python main.py
